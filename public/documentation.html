Documentation for My Quick Movie API v1.0.0


This is a project where users can access information about different movies, directors, and genres via this API.
Users will be able to sign up, update their personal informaiton, and create a list of their favorite movies and modify it.

Supported Requests

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies /movies GET None An array of JSON objects holding data about all the movies in the database
Return data about a single movie by title /movies/[title] GET None A JSON object holding data about the requested movie like this:
{'Title': 'The fountain',
'Description':'As a modern-day ...',
'Genre.Name':'Drama',
'Genre.Description':'In film ...',
'Director.Name': 'Darren Aronofsky',
'Director.Bio': 'Darren ...',
'Director.Birth': '1969',
'ImageURL': 'http ...',
'Featured': false}
Return data about a genre by name /movies/genre/[genre name] GET None An array of JSON objects holding data about the requested genre
{'Name':'Drama',
'Description': 'In film and...'}
Return data about a director /movies/directors/[director name] GET None An array of JSON objects holding data about the requested director
{'Name': 'Darren Aronofsky',
'Bio': 'Darren ...',
'Birth': 1969}
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:

{ 'Name': 'Mary Smith',
'favoriteMovies': []} 		A JSON object holding data about the user that was added, including their assigned id:
{'Name': 'Mary Smith',
'favoriteMovies': '[]',
'id': '#...',}
Allow users to update their user info (username) /users/[id] PUT A JSON object holding data about the user to add, structured like:

{ 'Name': 'Updated Name',
'favoriteMovies': []} 		A JSON object holding data about the user that was added, including their assigned id:
{'id': '#',
'Name': 'Updated Name',
'favoriteMovies': [],}
Add a movie to a user's favorites list by Title /users/[id]/favoriteMovies/[movieTitle] PATCH None A text message indicating whether the movie was successfully added to the user's list of favorite movies
Remove a movie from a user's favorites list /users/[id]/favoriteMovies/[movieTitle] DELETE None A text message indicating whether the movie was successfully removed from the user's list of favorite movies
Allow existing users to deregister /users/[id] DELETE None A text message indicating whether the user was successfully removed.
