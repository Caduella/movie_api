Documentation for My Quick Movie API v1.0.0


This is a project where users can access information about different movies, directors, and genres via this API.
Users will be able to sign up, update their personal informaiton, and create a list of their favorite movies and modify it.

Supported Requests

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies /movies GET None An array of JSON objects holding data about all the movies in the database
Return data about a single movie by title /movies/[title] GET None A JSON object holding data about the requested movie like this:
{'_id': '65aae20568a918670a575688' 'Title': 'The fountain',
'Description':'As a modern-day ...',
'Genre.Name':'Drama',
'Genre.Description':'In film ...',
'Director.Name': 'Darren Aronofsky',
'Director.Bio': 'Darren ...',
'Director.Birth': '1969',
'ImageURL': 'http ...',
'Featured': false}
Return data about a genre by name /movies/genre/[genre name] GET None An array of JSON objects holding data about the requested genre
{'Name':'Drama',
'Description': 'In film and...'}
Return data about a director /movies/directors/[director name] GET None An array of JSON objects holding data about the requested director
{'Name': 'Darren Aronofsky',
'Bio': 'Darren ...',
'Birth': 1969}
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:

{ 'Username': 'sdavis07',
'Password': 'Sdavis_2007',
'Email': 'sdavis07@example.com',
'Birthday': '2007-02-21'} 		A JSON object holding data about the user that was added, including their assigned id:
{'Username': 'sdavis07',
'Password': 'Sdavis_2007',
'Email': 'sdavis07@example.com',
'Birthday': '2007-02-21',
'FavoriteMovies': [],
'_id': '65ad66...'}
Allow users to update their user info /users/[username] PUT A JSON object holding data about the user to add, structured like:

{'Username': 'sdavis07',
'Password': 'Updated info',
'Email': 'sdavis07@example.com',
'Birthday': '2007-02-21'} 		A JSON object holding data about the user that was added, including their assigned id:
{'_id': '65ad66...',
'Username': 'sdavis07',
'Password': 'Updated info',
'Email': 'sdavis07@example.com',
'Birthday': '2007-02-21',
'FavoriteMovies': []}
Add a movie to a user's favorites list by Title /users/[username]/movies/[movieID] POST None A JSON object holding data about the user including the added movieID in the FavoriteMovies array.
Remove a movie from a user's favorites list /users/[username]/movies/[movieID] DELETE None A JSON object holding data about the user with the FavoriteMovies array now missing the deleted movieID.
Allow existing users to deregister /users/[username] DELETE None A text message indicating that [username] was deleted.
